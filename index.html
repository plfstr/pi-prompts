Pi Prompts

Basic Commands

startx
Access the Graphical User Interface (GUI) from the command prompt after the Pi has completed bootup.
rasp-config
Open the Pi configuration panel. Here you can overclock, make screen changes etc.
apt get update
Refresh the list of installed packages, so your Pi knows what is installed. Important Do this before performing an upgrade!
apt get upgrade
Performs an upgrade of the entire systems packages, collecting new files and their respective dependancies.
top
Display real-time system info for all currently running programes.
clear
Clears the terminal.
man example-command
Enter a command after man to show instructions for that command.
sudo halt -h
Shutdown the Raspberry Pi.
